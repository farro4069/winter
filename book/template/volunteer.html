Volunteer information

Volunteers are a necessary part of running an event such as this. There are a number of roles required, some have been allocated however further assistance would be appreciated.

    Roles include

  • Driver for lead vehicle
  • Driver for sag wagon
  • Event photographer
  • Other

If you would like to participate in the event please register for a place and select "volunteer" from the dropdown selector.

After you have registered someone will be in touch (initially by email) to discuss how you would like to be involved.
